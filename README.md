# Secure REST API Task Manager

This project is a secure REST API built with Flask and SQLite for managing tasks. It demonstrates backend development, REST API design, CRUD operations, input validation, basic token-based authentication, request logging, database usage, and Docker containerization.

The goal of this project is to practice building a structured backend application with security-oriented features such as request validation, input validation, protected endpoints, and parameterized database queries.

---

## Features

- Flask REST API
- SQLite database
- CRUD operations for tasks
- Token-based authentication using Bearer token
- Input validation for request data
- Request logging
- Parameterized SQL queries
- Error handling with proper HTTP status codes
- Dockerfile for containerization
- Clear project documentation and testing steps
